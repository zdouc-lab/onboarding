# Instructions

*Nota bene: assumes a fresh Ubuntu install*

## Installation

- Login, create new user, set password. Do not give sudo rights.

- Download and run the installer script
```commandline
wget https://raw.githubusercontent.com/zdouc-lab/onboarding/refs/heads/main/software_setup/setup_student.sh
sudo ./setup_student.sh
```

- Via the newly installed JetBrains Toolbox, install Pycharm (Community Version)


## Configuration

- [ ] Create a GitHub account and try to clone [modern_python](https://github.com/zdouc-group/modern_python) to your machine
- [ ] Setup Obsidian lab notebook
- [ ] Setup Element/Matrix.org
