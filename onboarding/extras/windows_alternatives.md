# Zdouc Laboratory Windows Alternatives

<img src="https://github.com/zdouc-lab/.github/raw/main/profile/zdouc_logo_v1.svg" style="width: 25vw"/>

*Nota bene: this is a living document and will change with time. Make sure to return regularly.*

The Zdouc lab is a Linux only lab. 
Therefore, many Windows programs will not be natively available for your work.
However, there is (almost) always an alternative, even if it means that approaches have to be adjusted.
For reference, Mitja has been working exclusively with Ubuntu Linux since 2011.

Here is an incomplete list of free (open source) programs usable for our research, sorted after topics. 
Feel free to search the web for additional alternatives and update this document.

## Writing & Figures

| Topic            | Alternative                                      |
|------------------|--------------------------------------------------| 
| Word processor   | [Google Docs](https://docs.google.com/)          |
| Citation manager | [Paperpile plugin](https://paperpile.com/)       |
| Figures          | [Inkscape](https://github.com/inkscape/inkscape) |
| Note-taking      | [Obsidian](https://obsidian.md/) |
